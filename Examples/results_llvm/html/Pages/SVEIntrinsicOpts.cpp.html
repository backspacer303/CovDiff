203 /// scalable vector of i1s, i.e. . ----
204 /// ----
205 /// The SVE ptrue intrinsic is used to create a logical representation of an SVE ----
206 /// predicate. Suppose that we have two SVE ptrue intrinsic calls: P1 and P2. If ----
